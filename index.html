Explaining the code

The Java File was created following these steps: I need to get 24000 samples of every musical note to can play it on the NIOS II processor on DE2 115 board. First at all I search for the frequencies in 4thoctave at this page: http://en.wikipedia.org/wiki/Piano_key_frequencies I wrote in an array of datatype double the frequences from C4 to B4, after that I learned some wave physics to obtain every sample from 0 to 23999, Asin(2PITIMEfrequency) where frequency has the same value for every single note and TIME is the value is changing through the loop. Then I created a file and wrote the notes on it.

Main Program initialices the interruption from buttons and keys, reads a String from memory and writes it on the JTAG UART port, after that does nothing.

HANDLEREXCEPTION: Detects the begining of the interruption and fix it using one routine or the other one, if a button was pressed, plays a song showing in the displays Playing this song: "name of the song". The displays are 7segment displays and are located at 0x10000020 and 0x10000030. Audio Port is located at 0x10003040 and JTAG UART Port is located at 0x10001000. And that's all I think, remember to enable interruption and set to 0 the bit that represents an interruption was launched and if you have doubts about it, ask me.
